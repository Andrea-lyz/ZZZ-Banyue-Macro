# 图片转文字，表格形式整理般岳技能信息

下面是按技能分类，把般岳技能从图片/资料整理成表格的文字版（以名称＋触发方式＋效果为主，方便你做键位或宏设计）。[^1]

## 普通攻击

| 技能名 | 触发条件 / 操作 | 主要效果简介 |
| :-- | :-- | :-- |
| 普通攻击：峥嵘 | 点按普攻（A款连段） | 最多四段前方火属性攻击，期间抗打断提升。[^1] |
| 普通攻击：崔巍 | 点按普攻（B款连段） | 最多四段前方火属性攻击，期间抗打断提升。[^1] |
| 普通攻击：焚身 | 不在怒相、嗔火≥120，长按普攻 | 对周身造成火伤并进入「怒相」状态。[^1] |
| 普通攻击：引火 | 非接战、HP＞50%、嗔火＜120，长按普攻 | 主动扣血回嗔火，嗔火到120直接进入怒相；进场赠送一次使用机会；最多扣至50%最大生命。[^1] |
| 普通攻击：倾山 | 怒相中，长按普攻消耗120嗔火 | 召唤巨大山体造成高火伤，结束后退出怒相，全程无敌。[^1] |
| 普通攻击：摧岳 | 怒相中，已通过普攻/指令释放过两种不同强化特殊技后，长按普攻 | 向前打出珠子禁锢并造成高火伤，然后自动接「倾山」，全程无敌。[^1] |

## 闪避 / 反击 / 冲刺

| 技能名 | 触发条件 / 操作 | 主要效果简介 |
| :-- | :-- | :-- |
| 闪避：不动如山 | 攻击中不推摇杆点按闪避 | 进入防御姿态，可招架攻击并积累失衡；命中瞬间被打为完美格挡，招架成功时无敌。[^1] |
| 闪避：越岭 | 点按闪避 | 普通冲刺闪避，期间无敌。[^1] |
| 闪避：叫阵 | 拖摇杆半周＋点按闪避 | 向后闪避并嘲讽；任意强化特殊技后可接叫阵，并在期间快速键入继续派生强化技，期间无敌。[^1] |
| 冲刺攻击：飞砂 | 闪避中点按攻击，或叫阵中长按攻击 | 向前抛珠造成火伤，可直接接峥嵘/崔巍的第二段。[^1] |
| 闪避反击：扬砾 | 越岭或叫阵触发极限闪避后点按/长按攻击 | 向前反击造成火伤，可直接接峥嵘/崔巍第三段，全程无敌。[^1] |

## 支援技

| 技能名 | 触发条件 / 操作 | 主要效果简介 |
| :-- | :-- | :-- |
| 快速支援：磐岩 | 队友被击飞时点按支援键 | 冲到前方连续攻击，造成火伤并无敌。[^1] |
| 招架支援：铁壁 | 前场角色即将被攻击时点按支援键 | 招架攻击，积累大量失衡，期间无敌。[^1] |
| 支援突击：昂霄 | 触发招架支援后点按攻击/支援键 | 快速突进攻击，火伤且无敌。[^1] |
| 支援突击：冲霄 | 通过「不动如山」成功招架后点按攻击/支援键 | 快速突进攻击，火伤且无敌。[^1] |

## 强化特殊技（含派生条件）

| 技能名 | 触发条件 / 操作 | 主要效果简介 |
| :-- | :-- | :-- |
| 强化特殊技：山摇 | 闪能足；崔巍1、2段后点按，或任意强化技/焚身/连携/终结/支援突击/闪避反击/快速支援中快速键入指令 | 掀山体造成火伤，全程无敌。[^1] |
| 强化特殊技：地动 | 闪能足；峥嵘3、4段后点按，或支援突击/闪避反击/快速支援后长按，或上述招式期间快速键入指令 | 连续重击地面造成火伤，无敌。[^1] |
| 强化特殊技：狮子吼 | 闪能足；峥嵘1、2段后点按，或任意强化技/焚身/连携/终结/支援突击/闪避反击/快速支援中快速键入指令 | 前方音波火伤，无敌。[^1] |
| 强化特殊技：论道 | 闪能足；崔巍3、4段后点按，或支援突击/闪避反击/快速支援后长按，或上述期间快速键入指令 | 投掷珠子远程火伤，无敌。[^1] |
| 强化特殊技：狮子吼·怒 | 闪能足；在释放「论道」时快速键入指令 | 更强力音波火伤，无敌。[^1] |
| 强化特殊技：山摇·怒 | 闪能足；在释放「地动」时快速键入指令 | 更巨大的山体火伤，无敌。[^1] |
| 取消 / 完美取消 | 快速键入派生强化技时中断当前强化技（在各自完美打断节点） | 完美节点后打断不损伤害；在完美节点打断判定为「完美取消」，怒相＋有山威时更易触发。[^1] |

## 连携技 / 终结技

| 技能名 | 触发条件 / 操作 | 主要效果简介 |
| :-- | :-- | :-- |
| 连携技：怒焰 | 触发连携时选择般岳 | 前方大范围火伤；若嗔火≥120，结束后自动进入怒相，全程无敌。[^1] |
| 终结技：撼天动地 | 喧响等级达到「极」时点按终结技 | 大范围火伤；嗔火≥120则结束后自动进入怒相，全程无敌。[^1] |

| 项目 | 数值 / 规则 | 说明 |
| :-- | :-- | :-- |
| 嗔火上限 | 150 | ≥120可焚身入怒相。[^1] |
| 山威获取 | 进入怒相时获得4层 | 有山威时强化特殊技改为消耗山威并回10闪能，不耗闪能；退出怒相清空。[^1] |
| 嗔火获得 | 每1点闪能→0.5嗔火；招架支援＋不动如山/极限闪避等额外回复 | 具体数值：招架支援+4，招架成功+4，完美格挡+6，极限闪避+4，每秒一次。[^1] |
